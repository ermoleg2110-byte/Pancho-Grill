# Pancho-Grill
## Что это?
Этот проект - сайт для ресторана Pancho Grill, созданный в целях обучения работы с языками программирования, такими как : HTML, Python, CSS, а также фреймворк Django.
## Как запустить проект?
Для запуска проекта поместите папку **pancho grill** в среду программирования, и в терминале пропишите:
**python pancho grill/manage.py runserver**
После перейдите в браузер по ссылке **http://127.0.0.1:8000/**, и вы попадете на сайт ресторана.
